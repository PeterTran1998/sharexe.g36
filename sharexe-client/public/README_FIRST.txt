Các file trong các thư mục css, fonts, js và webfonts là các thư viện bên ngoài, không nên đụng tới nó, chỉ xài thôi
	- fontawesome của template mình đang xài là bản 4.7

File custom-style.* là các file bổ sung cho file style.css, nó chưa một số định dạng cho các element chưa được hiện thực trong template, chẳng hạn modals
File only-for-index.css chứa css chỉ sử dụng cho trang intro (index.html)
Chỉ chỉnh sửa và bổ sung cái file không nằm trong thư mục con (style.css, custom-style.*)
